집잇 개인정보처리방침

시행일: 2026년 3월 23일

집잇(이하 "서비스")은 이용자의 개인정보를 중요시하며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 개인정보처리방침은 서비스가 수집하는 개인정보의 항목, 수집 목적, 보유 기간 등을 안내합니다.

1. 수집하는 개인정보 항목

수집 항목 수집 목적 보유 기간
이메일, 비밀번호(암호화) 회원가입 및 로그인 회원 탈퇴 시까지
닉네임 서비스 내 표시 회원 탈퇴 시까지
무주택 기간, 부양가족 수, 세대주 여부, 혼인 여부, 청약통장 가입기간, 납입 횟수 청약 당첨 가능성 분석 회원 탈퇴 시까지
계약 정보(주소, 아파트명, 가격, 계약일정) 부동산 계약 관리 기능 제공 회원 탈퇴 시까지

2. 기기 내 저장 정보

아래 정보는 서버에 전송되지 않으며, 이용자의 기기에만 저장됩니다.

  • 관심 설정 (관심 유형, 지역, 예산 범위)
  • 위젯 설정
  • 알림 이력 (최대 50건)

3. 수집하지 않는 정보

  • 위치 정보 (GPS)
  • 기기 고유 식별자
  • 광고 ID
  • 연락처, 사진, 카메라
  • 분석 및 추적 데이터 (Analytics 미사용)

4. 개인정보의 제3자 제공

서비스는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 외부 서비스를 이용합니다.

  • Supabase: 회원 인증 및 데이터 저장 (AWS 서버)
  • 국토교통부 공공데이터 API: 실거래가 조회 (개인정보 전송 없음)
  • Kakao API: 주소 검색 (검색어만 전송, 개인정보 아님)

5. 개인정보의 파기

회원 탈퇴 시 서버에 저장된 모든 개인정보는 지체 없이 파기합니다. 기기에 저장된 정보는 앱 삭제 시 자동으로 삭제됩니다.

6. 이용자의 권리

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

  • 개인정보 열람, 수정, 삭제 요청
  • 회원 탈퇴를 통한 개인정보 삭제
  • 개인정보 처리 정지 요청
